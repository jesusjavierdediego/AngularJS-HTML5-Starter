AngularJS App Starter
=====================

### Quick Start

#### Before you start, tools you will need
* install npm
* bower and grunt (run the following commands):

```script
npm install -g bower
npm install -g grunt
```

## Running
* configure project:

```script
npm install
bower install
```
* run project

`grunt server`

## Testing

Exexecute the following command to launch tests

`grunt test`
